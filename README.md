# 📝 Smart Notes App

A modern and responsive Notes Application built using React.js. This project allows users to create, edit, delete, search, and manage notes efficiently with local storage support.

## 🌐 Live Demo

👉 https://your-netlify-link.netlify.app

## ✨ Features

- Create new notes
- Edit existing notes
- Delete notes
- Search notes instantly
- Responsive design
- Local Storage support
- Clean and user-friendly interface

## 🛠️ Tech Stack

- React.js
- JavaScript
- HTML5
- CSS3
- Vite
- Local Storage API

## 🚀 Installation

```bash
npm install
npm run dev
```

## 📦 Production Build

```bash
npm run build
```

## 🎯 Concepts Implemented

- React Components
- JSX
- useState Hook
- Event Handling
- Conditional Rendering
- List Rendering
- CRUD Operations
- Search Functionality
- Local Storage Integration

## 🚀 Deployment

Deployed using Netlify.
